Multinomial–Dirichlet Bayesian Updating (베이즈 갱신)

True Probability (실제 확률)을 설정하고, Observation (관측)이 누적될수록 Posterior Distribution (사후분포)이 어떻게 변화하는지 직접 확인하세요.

True Probability (실제 확률) θ
Sum (합) = 1.000
Prior (사전분포) α — Dirichlet
Roll Dice (주사위 굴리기)
Total Observations (총 관측): 0
Observation Count (관측 횟수)
Posterior Distribution (사후분포)
Convergence (수렴 과정)
Conjugate Prior (켤레 사전분포) Relationship (관계)
Likelihood (가능도): Multinomial(n; θ₁,…,θ₆)
Prior (사전분포): Dirichlet(α₁,…,α₆)
Posterior (사후분포): Dirichlet(α₁+n₁,…,α₆+n₆)

Posterior Mean (사후 기대값): E[θᵢ] = (αᵢ+nᵢ) / Σ(αⱼ+nⱼ)
Observation (관측)이 누적될수록 Posterior (사후분포)가 True Probability (실제 확률) θ로 Converge (수렴)하는 것을 확인하세요.
